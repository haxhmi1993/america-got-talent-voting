Test Assignment
Build a simplified “America’s Got Talent” voting system. The front end presents a single text input where a user enters a contestant’s last name and submits a vote.
Frontend Scope
One text input and a submit button
Display success or error only
Backend Scope
Backend must enforce vote integrity and prevent casual fraud. It must accept a text vote, normalize and validate the candidate, generate or verify a server-issued token tied to a hashed device fingerprint, and track votes so each device can vote once per candidate and no more than three total. It must detect and throttle suspicious repeat attempts via IP-aware rate limiting and short-term caching and fall back to CAPTCHA or SMS verification for high-risk cases.
